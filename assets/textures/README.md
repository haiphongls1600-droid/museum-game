# Textures Pack

Thư mục này chứa toàn bộ texture (ảnh vẽ) dùng trong game museum.

## Quy ước
- Ảnh nền phòng
- Tường
- Sàn
- Kệ trưng bày
- Cây cối, vật trang trí

## Gợi ý cấu trúc
- floor.png        : texture sàn
- wall.png         : texture tường
- shelf.png        : kệ trưng bày
- plant.png        : cây / cỏ
- decoration.png  : vật trang trí khác

## Lưu ý
- Ưu tiên PNG
- Nền trong suốt nếu là vật thể
- Kích thước nên là bội số của 16 hoặc 32
